Typography

Heading Font: Amaranth

I picked this font because it looks good on the web, while still creating a professional look for a business. It pairs well with open sans, which is a very popluar font for web use, and it's different and unique, which I wanted to help this site stand out a little bit.

Paragraph Font: Open Sans

I chose this font because it's very readable, and one of the most common fonts on the web. I like that it looks professional and it's easy on the eyes when people are reading text on the screen for a long time.
